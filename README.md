# 📊 Cognifyz Technologies – Data Analysis Internship Project

## 📌 Project Overview
This project was completed as part of the Cognifyz Technologies Data Analysis Internship.  
The objective was to analyze a restaurant dataset and extract meaningful business insights using Python.

---

## 🎯 Business Objectives
- Identify top-performing cuisines
- Analyze city-wise restaurant distribution
- Study price range patterns
- Examine impact of online delivery on ratings
- Explore correlation between votes and ratings
- Identify popular restaurant chains

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
